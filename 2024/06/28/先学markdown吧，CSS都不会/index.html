开始之前需要有几个注意事项：

  1. 要创建段落，请使用空白行将一行或多行文本进行分隔。
  2. Markdown 本身并不直接支持改变文字颜色或其他排版细节如字体和大小。Markdown的设计初衷是保持文本的简洁性和易读性，重点在于结构化内容而不是样式。然而，如果你需要在Markdown文档中改变文字颜色，可以借助内嵌HTML的方式来实现。由于许多Markdown解析器也支持HTML代码，你可以使用HTML的标签配合style属性来改变文字颜色，例如：<span style="color: red;">这段文字将会是红色</span>。于是它就变成这样：这段文字将会是红色（HTML学习也要提上日程了）

粗体和斜体

粗体在要加粗的内容前后加两个*，斜体只要一个。

注意：** 符号用于加粗文本，Markdown语法不能加粗！
比如~
粗体： **我变粗了吗**
斜体： *我斜了吗*
那么它就会显示
粗体： 我变粗了吗
斜体： 我斜了吗

删除线

若要删除内容，请在单词前后使用两个波浪号 。
`这句话被删除了`
~~这句话被删除了

标题

注意：用一个空格在 # 和标题之间进行分隔.
# 这是大标题
## 这是中标题
### 这是小标题
#### 这是小小标题
##### 这是小小小标题
###### 这是小小小小标题

引用

我们用“>”表示引用，不过具体格式在不同的编辑器和网站上差异较大
比如~
> “真正的学者知道怎样从已知引出未知，并且逐步接近于它。” —— 歌德
结果是这样子滴~

“真正的学者知道怎样从已知引出未知，并且逐步接近于它。” —— 歌德

代码

那作为敲键盘的牛马，代码怎么显示呢？
这里有两种方法：行内代码和代码块

行内代码需要使用` `是这样的`int a = 10; float b = 0.1f`

代码块需要``` ```框定起来
显示是这样的~
int a = 10;
int b = 20;
int c = a + b;

链接和图片

插入图片Markdown语法代码:![图片alt](图片链接 "图片title")

图片alt：当图片因为某些原因无法显示时（比如图片链接失效、网络问题或用户使用的是屏幕阅读器等辅助技术），alt 文本就会显示出来

比如:
![这是个美女图片](//Users/fang/blog/source/OIP-C.jpeg "嘿嘿 好看")

列表

列表分为有序列表和无序列表
无序列表的是加一个”-“

  • a
  • b
    • b1
    • b2
  • c
    有序列表就简单了
  1. a
  2. b
  3. c

分割线

想要在两行之间加入分割线，最简单的方法就是：新起一行输入***
比如~
abcdefghijklmnopqrstuvwxyz
***
abcdefghijklmnopqrstuvwxyz
结果就会在两行abc中间加一条分割线
abcdefghijklmnopqrstuvwxyz

abcdefghijklmnopqrstuvwxyz

转义

要显示原本用于格式化 Markdown 文档的字符，请在字符前面添加反斜杠字符 \ 。
当我们使用 \>hello world 时
>hello world 此时引用符号无效了。

待办事项

我们使用- [ ] 来创建待办事项，做了哪个就在哪个框子里面画叉号
比如：
- [ ] 好好学习
- [x] 天天睡觉
它的显示结果就是这样的：

  • 好好学习
  • 天天睡觉

最后，也是最重要的，由于 Markdown 本质上是 HTML 的一种别名，那么在 Markdown 中直接书写 HTML
就成了非常自然的事情。
